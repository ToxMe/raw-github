Phobos

A cdn for github, designed for those who commit

Totally free

Both free as in cost and speech, Phobos is licenced Apache v2 and costs exactly $0 to use for everyone of all shapes and sizes. use it for production, for development, we place 0 restrictions on you.

Try it out

Faster than Github alone

First we take what's provided by Github, then we remove almost half a kb worth of unneeded headers, passing it on to be minified. Afterwards the data is cached on Cloudflares network of 64+ datacenters, boasting an average response time of 30 ms.

Instantly view changes

We respect Githubs headers, letting us replace the cached version with the latest and greatest version of your changes.
